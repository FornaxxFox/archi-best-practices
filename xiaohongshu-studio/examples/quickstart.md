# Quickstart · v1.3 三步走

> 第一次用本插件？按这个顺序跑一圈，30 分钟内感受 v1.3 的全部新能力。

## Step 0 · 初始化（首次使用）

不需要配置文件，不需要 API key，直接进入 Step 1。

## Step 1 · 跑通基础流水线（5 分钟）

对插件说：

> 帮我拆《深度工作》做小红书图文

编排器会自动跑：原文摄入 → 选题策划 → 文案撰写 → 海报生成 → 封面插图 → 责编审校 → 主编审阅 → 观众视角 → 发布物打包。

**唯一要拍板的是 Stage 2**：从 3–5 个候选角度里选 1 个。

完成后看 `projects/拆书-深度工作/outputs/`，里面应该有：
- `*-小红书海报.pptx`
- `*-发布文案.md`
- `*-审校记录.md`
- `*-发布指引.md`

导出 PNG 就能发。

## Step 2 · 建立人设语料库（10 分钟）

对插件说：

> 跑一下人设语料库

然后贴 3–5 篇你自己写过的内容（公众号 / 小红书 / 朋友圈金句都行）。

插件会：
1. 提取你的口头禅 / 句式 / emoji 密度 / 标题钩子偏好
2. 列 5–8 个你"绝对不说的话"
3. 生成 `references/persona-profile.json`

完成后，下次跑 `/文案撰写` 时会**自动**做人设一致性检查。命中 forbidden_phrases 直接 0 分回炉。

**这步推荐每个新账号做一次**。已经做过的可以跳过，persona 会自动跨项目生效。

## Step 3 · 做系列化内容（15 分钟）

对插件说：

> 用深度工作做系列化规划，8 期

插件会生成 `projects/拆书-深度工作/series-bible.md`，包含：
- 8 期排期表（每期副标题 + 钩子 + 内容骨架 + 与上期关系）
- 节奏表（每周二 19:30）
- 跨期引用机制（承接句 + 预告句）
- 累计涨粉预估

然后你可以说：

> 做 Vol.1

或：

> 跑完整个排期表

编排器会自动按排期表跑所有期，**承接/预告句自动注入**。

## Step 4 · 多平台一鱼多吃（10 分钟）

打包阶段勾选"多平台适配"，或单独说：

> 跑多平台适配

插件会基于小红书发布包生成 4 个平台版本：
- **抖音图文**：≤300 字 / 9:16 / 抖音热词
- **公众号**：1500–2000 字 / 章节小标题 / 1:1 头图
- **知乎**：1500–2500 字 / 问答体 / 补论据
- **即刻 + Twitter**：≤280 字 / 短金句

发布指引里会附上**错峰表**：
- 小红书 + 即刻 + Twitter：周二 19:30
- 抖音图文：周三 19:30（错峰 24h）
- 公众号：周四 19:30（错峰 48h）
- 知乎：周五 19:30（错峰 72h）

## Step 5 · 数据闭环（24h+ 后发布）

发布后回来说：

> 复盘上期

插件会：
1. 收集平台数据（手动输入 / 截图 / API）
2. 写入 `publish-history.jsonl`（含 platform / series / persona_score 字段）
3. 更新 `historical-insights.md`（Top 3 / Anti-pattern / Audience signals）
4. 触发 persona 漂移告警（如连续 3 期 < 3.5）
5. 跨平台聚合（如启用了多平台）

下次跑选题时，这些数据会自动影响推荐。

## 三个 skill 的关系

| 想要 | 跑这个 |
|---|---|
| 一次发 1 篇 | `/编辑部编排` |
| 一次发 5–10 篇（系列） | 先 `/系列化规划` 再 `/编辑部编排` |
| 一篇发 6 个平台 | `/多平台适配`（打包阶段或独立跑） |
| 保持"我的语感" | `/人设语料库`（首次必跑） |
| 看数据怎么影响下一期 | `/发布复盘`（发布 24h+ 后） |

## 常见问题

**Q: 不做系列可以跑 v1.3 吗？**
A: 可以。所有 v1.2 行为完全保留，新增能力是 opt-in。

**Q: 一定要先做人设吗？**
A: 不做也行，persona-profile.json 缺失时文案撰写跳过一致性检查。但强烈建议做一次，能避免"通用 AI 味"。

**Q: 多平台版本可以分开发吗？**
A: 可以。每个平台是独立 .md 文件，按错峰表自己选时机。

**Q: 系列跑一半想改方向怎么办？**
A: 改 series-bible.md，然后说"按新排期表继续"。老的 vol 不变。

**Q: 公众号不想做，能跳过吗？**
A: 能。`/多平台适配` 会问"哪些平台"，只勾你要的。
