# 三读者剧本卡

用于「观众视角」skill 生成真实反应。**关键：写内心 OS，不写分析。**

## 读者 A · 3 秒刷客 · 剧本

**人设**：25 岁互联网打工人，晚上 10 点躺床上刷小红书。手机拿右手，拇指飞快。手上还端着刚喝了一口的酸奶。

**内心 OS 模板**：

```
[看到首图 0.3 秒]
"啊？"  / "呃..."  / "哦哟" / "什么鬼" / "..."（继续滑）

[如果停下 3 秒]
"这是啥？"  / "跟我有关吗？"  / "有点意思"

[进入第 2 页]
"字有点多啊..." / "哦，我确实这样"

[中途]
"值得存"  / "感觉又是那套" / "跳走"

[尾页]
"存了"  / "点关注"  / "算了下次一定"
```

**必须给出**：
- 首图第一反应（一句短句）
- 停留时长（几秒）
- 跳出/继续的具体节点
- 最终动作（关闭/收藏/关注）

## 读者 B · 深度学习党 · 剧本

**人设**：30 岁产品经理，最近在焦虑"如何变得更专注"。周六下午泡在咖啡馆里找干货。

**内心 OS 模板**：

```
[第 1 页]
"标题挺戳我" / "又是这种老话题"

[第 2-3 页]
"哦这个说法我第一次听" / "老生常谈了"

[第 4-5 页]
"这个公式有点意思" / "这几种分类之前 XX 说过"

[第 6-7 页]
"数据靠谱吗" / "有说服力" / "缺依据"

[第 8 页]
"这个我今晚就能试" / "太笼统了没法执行"

[第 9 页]
"这句话不错" / "收藏"

[整体]
"值不值 5 分钟" / "会不会给同事推荐"
```

**必须给出**：
- 每页的"学到了什么" or "已经知道"
- 具体想执行的 action
- 会不会评论、评论什么
- 是否关注

## 读者 C · 挑刺党 · 剧本

**人设**：40 岁资深内容人，见过太多套路。看内容像看合同——找漏洞、找卖点、找雷点。

**内心 OS 模板**：

```
[标题]
"广告法擦边" / "还行没踩线"

[数据/引用]
"来源呢" / "这是原文吗" / "有断章取义嫌疑"

[结构]
"这不就是 XX 的爆款模板" / "有独立思考"

[金句]
"鸡汤味重" / "确实精辟"

[整体]
"能过审但会被同行嘲" / "值得转发" / "路人局，不推荐"
```

**必须给出**：
- 挑出的具体问题（页码+具体表述）
- 认可的部分
- 综合判断："能发"/"改后发"/"不建议发"

## 反应生成守则

1. **短句、口语化**：像发朋友圈一样写，不像写报告
2. **保留情绪词**：`啊、哦、呃、卧槽（雅化用"哎"）、算了、可以`
3. **允许自相矛盾**：真实读者会突然反悔（"想收藏 → 算了下次一定"）
4. **允许停顿**：`...` `嗯...` `等等`
5. **不用"我作为读者觉得"**——直接就是"我"

## 反面示范（禁止）

- ❌ "读者可能会觉得这篇内容有一定的吸引力，但仍需优化。"
- ❌ "从数据角度看，用户停留时长预计中等偏上。"
- ❌ "建议在首图部分强化视觉冲击力。"

## 正面示范

- ✅ "哦哟，'越忙越没成就感' — 说的不就是我 —— 停下来看看"
- ✅ "P3 字有点多啊...算了跳一下...哎 P4 有个公式 —— 拍照存了"
- ✅ "P6 说'研究显示' — 谁的研究啊，扣分"

## 汇总时的写法

三读者的反应汇总成"热力图"，需要具体到页：

```
P1  A: ⭐⭐⭐⭐⭐（停下） B: ⭐⭐⭐⭐（进入） C: ⭐⭐⭐（可用）
P4  A: ⭐⭐⭐（模糊瞄了眼） B: ⭐⭐⭐⭐⭐（爱了） C: ⭐⭐⭐⭐
P6  A: ⭐⭐⭐⭐（数字抓眼球） B: ⭐⭐⭐（想追问） C: ⭐（要来源）
...
```

哪一页三个读者都是 ⭐⭐⭐ 以下 → 那页要重做。
