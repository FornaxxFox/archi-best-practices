# 平台规格速查表

> 数据经验值（2025–2026），平台规则变动以官方为准。

## 小红书（基准）

| 字段 | 规格 |
|---|---|
| 字数（长版） | 350–450 字 |
| 字数（短版） | 120–180 字 |
| 图片比例 | 3:4 (1080×1440 推荐) |
| 图片数 | 1–9 张（推荐 9 张） |
| 标签数 | 8–12 个 |
| 标题字数 | ≤ 22 字 |
| 视频时长 | 60s 内 |

## 抖音图文

| 字段 | 规格 |
|---|---|
| 字数 | ≤ 300 字（图文字数） |
| 图片比例 | 9:16 (1080×1920) |
| 图片数 | 1–6 张（推荐 3–5 张） |
| 标签数 | 5–8 个热词 |
| 标题字数 | ≤ 30 字 |
| 视频版 | 60s 内竖屏 |

**与小红书差异**：
- 标签必须换抖音热门词（"挑战"、"热梗"、"话题"风格）
- 抖音用户更爱口语，emoji 可放更多
- 抖音更推"挑战体"和"反转"

## 公众号

| 字段 | 规格 |
|---|---|
| 字数 | 1500–2000 字（短文）或 3000+（深度） |
| 图片比例 | 1:1 (900×900) 或 16:9 |
| 图片数 | 3–6 张（头图 + 内文 + 末尾） |
| 标签 | 文末 "#话题" 或不留 |
| 标题字数 | ≤ 30 字（推送显示） |
| 摘要 | ≤ 54 字（推送列表） |

**与小红书差异**：
- 必须加章节小标题（H2/H3），方便扫读
- 必须配"封面大图"（900×383 横幅 或 1:1 方形）
- 文末必须有"关注引导"
- 公众号对"诱导分享"（如"求转发"）限流，禁用
- 医疗 / 金融 / 政治类内容需更严合规

## 知乎

| 字段 | 规格 |
|---|---|
| 字数 | 1500–2500 字（答案） |
| 图片比例 | 1:1 (800×800) 或宽屏 |
| 图片数 | 3–8 张 |
| 话题（问题） | 3–5 个相关话题 |
| 标题 | 问题本身，答案首行加粗 |
| 结语 | "以上" / "以上" + 互动 |

**与小红书差异**：
- 必须有"问 / 答"结构（即使原文没有，也要补一句"问题背景"）
- 必须补论据（论文、报告、案例），知乎用户对"无证据观点"反感
- 引用格式：使用 `> 引用` 或图片截屏
- 不允许"家人们"、"绝绝子"等小红书语
- 知乎对"医疗 / 金融"要求更严，需专业资质标识

## 即刻

| 字段 | 规格 |
|---|---|
| 字数 | ≤ 280 字 |
| 图片 | 1 张（推荐）或 9 张 |
| 比例 | 1:1 或 4:3 |
| 标签 | 1–2 个 # |
| 标题 | 无独立标题，首句即钩 |

**与小红书差异**：
- 文人社区氛围，文艺 / 思考 / 调侃 都行
- 短句节奏更适合（一行一个意思）
- 平台重视原创度，搬运感会被踩
- 评论文化深，可引导讨论

## Twitter (X)

| 字段 | 规格 |
|---|---|
| 字数 | ≤ 280 字（英文算字符不同） |
| 图片 | 1 张（推荐）或 4 张 |
| 比例 | 16:9 |
| 标签 | 1–2 个 # |
| 标题 | 无独立标题，首句即钩 |

**与小红书差异**：
- 中文 ≤ 280 字，英文 ≤ 280 字符
- 不允许诱导互动（"求点赞"会被降权）
- 标签必须英文为主（#productivity 等）

## 平台禁忌清单

| 平台 | 禁忌 |
|---|---|
| 小红书 | 极限词、夸大宣传、医疗承诺 |
| 抖音 | 搬运、纯文字无图、低质封面 |
| 公众号 | 诱导分享、标题党、P2P/荐股 |
| 知乎 | 无证据观点、AI 注水、营销号感 |
| 即刻 | 搬运水文、商业硬广 |
| Twitter | 同小红书 + 多语言混杂（影响分发） |
