# 爆款要素清单

审校要素打分表，每项 1–5 分。

## 1. 首图钩子（权重 25%）

**满分标准（5 分）**：
- 主标题 3 秒内能看懂"跟我有关"
- 有痛点/反差/悬念/数字，四中至少一个
- 字号足够大（40pt+），深浅对比明显
- 情绪 anchor 清晰（颜色/符号/emoji）

**扣分项**：
- 标题空泛（"干货满满"、"值得收藏"）→ 3 分
- 视觉杂乱、没有主视觉焦点 → 2 分
- 用了极限词或平台限流词 → 1 分

## 2. 情绪共鸣（权重 15%）

**满分标准**：
- 至少一处让读者"这说的就是我"的场景
- 用"你"而不是"大家"的口吻
- 有 emoji 情绪锚点，节奏合理

**扣分项**：
- 全程说教口吻 → 3 分
- 完全没有场景描写 → 2 分

## 3. 具体数字/公式（权重 15%）

**满分标准**：
- 至少 1 个具体数字/百分比/公式
- 数字放在了显眼位置（P4 或 P6）
- 有来源或合理约束（"据研究"、"约"）

**扣分项**：
- 全篇无数据 → 2 分
- 数据无来源、可能夸大 → 1 分（合规风险）

## 4. 可执行 action（权重 15%）

**满分标准**：
- 有明确的 action list 页
- 每个 action 具体到"多久/工具/怎么衡量"
- 用户看完能立刻做一件事

**扣分项**：
- Action 空泛（"多练习"、"多思考"）→ 2 分
- 完全没 action → 1 分

## 5. 收藏动机（权重 10%）

**满分标准**：
- 有可打包记忆的框架（X 条法则、X 步方法）
- 有值得截图保存的金句页
- 内容密度让用户"下次还想看"

## 6. 讨论钩子（权重 10%）

**满分标准**：
- 结尾有可评论的问题
- 评论区置顶话题清晰

**扣分项**：
- 结尾直接结束、无 CTA → 3 分

## 7. 视觉一致性（权重 10%）

**满分标准**：
- 9 页调色板一致
- 有跨页视觉母题（页码、装饰条）
- 字号层级清晰、无溢出

**扣分项**：
- 页面之间风格跳跃 → 3 分
- 有文字溢出/断字 → 2 分（严重视觉损失）

## 总分计算

```
总分 = Σ(每项分数 × 权重)
```

- ≥ 3.8 → 直接通过
- 3.0–3.8 → 条件通过（给具体建议）
- < 3.0 → 建议重做

## 快速判断法（10 秒版）

只看三件事就能判断能否发：
1. 首图 3 秒能不能勾住人？
2. 有没有一个具体的可 action 的东西？
3. 有没有踩合规红线？

前两个 yes + 第三个 no → 大概率能发。
