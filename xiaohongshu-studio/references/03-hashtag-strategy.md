# 标签策略

## 组合公式（8–12 个）

```
[大流量池 3 个] + [精准标签 4–6 个] + [长尾/小众 2–3 个]
```

## 大流量池（每篇必带 3 个，从下方挑）

| 领域 | 常用大标签 |
|---|---|
| 成长/职场 | #个人成长 #职场干货 #自律生活 #时间管理 #效率提升 |
| 阅读 | #读书笔记 #好书推荐 #拆书笔记 #每天一本书 #书单 |
| 心理 | #心理学 #自我疗愈 #情绪管理 |
| 生活 | #生活方式 #生活小技巧 |
| 学习 | #学习方法 #考研 #学习打卡 |

## 精准标签（4–6 个）

按内容主题挑选。**具体到书名/工具名/方法论名**：
- 书名：`#深度工作`、`#原子习惯`
- 方法论：`#番茄工作法`、`#GTD`
- 场景：`#打工人日常`、`#通勤时光`
- 状态：`#打工人自救`、`#下班后`

## 长尾/小众（2–3 个）

流量小但精准，用来打入细分圈子：
- 作者名：`#卡尔纽波特`
- 细分主题：`#专注力训练`
- 小众场景：`#副业成长`、`#独立开发`

## 排列顺序

**从大到小**：大流量池 → 精准 → 长尾。这样发布后前几个标签更容易进入大流量池。

## 上限与避雷

- 单篇 8–12 个最佳，超过 15 个可能被判定标签堆砌
- 避免和内容无关的标签（会限流）
- 避免政治/敏感话题标签
- 不要重复同义标签（比如 #专注 #专注力 只留 1 个）

## 快速自检

发布前问自己：
1. 有没有覆盖 1 个主话题的大标签？
2. 有没有 1 个能被目标读者搜到的精准标签？
3. 有没有 1 个可以进入垂类圈子的小众标签？

三个都有 → 通过。
