# 选题模板

## 单个角度结构

```markdown
### 角度 N：{一句话定位}

- **钩子标题**：{≤ 22 字}
- **首图关键词**：{视觉方向：暖色系 / 深色 hook / 数字对比…}
- **目标读者**：{具体到人群 + 场景}
- **情绪按钮**：焦虑 / 共鸣 / 好奇 / 优越 / 恍然大悟（选 1–2）
- **内容骨架**：9 页拆解
  - P1 封面钩子
  - P2 痛点铺垫
  - P3 概念澄清
  - P4 核心公式/机制
  - P5 分类/方法论
  - P6 数据卡（关键数字放大）
  - P7 法则/框架
  - P8 今晚就能做的行动
  - P9 金句 + CTA
- **爆款潜力评分**：⭐⭐⭐⭐⭐（1–5）
- **推荐理由**：{为什么这个角度会火}
- **风险点**：{合规/准确性/情绪导向}
```

## 钩子标题模板（可套用）

| 类型 | 模板 | 示例 |
|---|---|---|
| 痛点疑问 | 为什么你 {现象}？ | 为什么你越忙越没成就感？ |
| 反差数字 | X 小时 = Y 小时 | 4 小时 = 别人的 12 小时 |
| 自我打脸 | 我以为 {A}，其实 {B} | 我以为是懒，其实是深度不够 |
| 结果承诺 | X 天做完 Y | 30 天读完 12 本书的秘密 |
| 权威引用 | XX 教授说 | 麻省理工教授：这个公式最重要 |
| 情绪共鸣 | 每个 {人群} 都该看 | 每个打工人都该收藏这 9 张图 |

## 目标读者画像模板

- **职业身份**：如"25–35 岁一线城市白领"
- **典型场景**：如"下班回家累到只想刷手机"
- **情绪状态**：如"焦虑但没方向"
- **他们缺什么**：如"确定感 + 可执行方法"
- **他们收藏动机**：如"以后再看 / 转发给朋友证明我在成长"
