# 合规红线清单

## 一票否决项

### 1. 广告法极限词

**必删清单**：

```
最、第一、极致、绝对、唯一、顶级、独家、首创、开创性、
永远、永久、百分百、100%、包治百病、根治、彻底、完全、
最佳、最好、最强、最大、最全、最高、最新（后接产品/效果）、
国家级、世界级、宇宙级、殿堂级
```

**替换建议**：

- 最 → 很/非常/相当
- 第一 → 领先/靠前
- 绝对 → 通常/大多数情况下
- 百分百 → 大概率/几乎/据统计
- 最佳 → 推荐/优选

### 2. 医疗健康承诺

**红线**：

- 治愈、根治、彻底解决 XX 病
- 保证瘦身/增高/丰胸/生发
- 替代药物/替代医生

**处理**：
- 涉及健康话题一律加免责："以上仅为分享，非专业医疗建议，具体请咨询医生"
- 不能出现"治疗"、"疗效"、"处方"等词
- 涉及药品名一律驳回

### 3. 金融理财承诺

**红线**：

- 保本、稳赚、无风险、月入 X 万
- 具体推荐股票代码、基金代码
- 涉及虚拟货币、外汇、期货

**处理**：
- 谈钱一律加"投资有风险，决策需谨慎"
- 不能给出具体买卖建议

### 4. 敏感话题

**驳回类**：

- 政治敏感、涉台涉港涉疆
- 涉暴、涉黄、涉赌
- 未成年不当引导
- 民族/宗教/地域歧视
- 造谣、未经证实的负面信息

### 5. 数据/引用溯源

**必查**：

- 每个百分比、具体数字，是否有明确来源？
- 引用他人观点，是否署名？
- 数据是否是原文数据，还是二次加工的？

**处理**：
- 无源数据 → 加"约"、"据……估算"、"作者观点"
- 未署名引用 → 补上出处
- 存疑数据 → 驳回

### 6. 版权

**必查**：

- 使用的图片是否有版权风险？
- 引用的文段是否超过合理引用范围？
- 字体、素材、模板是否商用授权？

## 快速扫描模板

审校前先跑一遍关键词扫描（Bash 或 grep）：

```bash
grep -E "最|第一|绝对|唯一|百分百|100%|包治|根治|彻底" copy/*.md
```

命中一律停下核对。
